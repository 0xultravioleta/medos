### Architecture Highlights
The master plan defines 8 core modules (A through H) covering the full healthcare operations stack. Key technical decisions:
- **FastAPI + Python 3.12** for backend (AI/ML ecosystem)
- **Next.js 15** for frontend (Server Components for PHI safety)
- **PostgreSQL 17 + pgvector** as single DB for relational + FHIR + vectors
- **Claude API** as primary LLM (HIPAA BAA available)
- **Terraform** for all infrastructure (never CloudFormation)
